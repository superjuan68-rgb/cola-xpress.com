Export Brand Assets to PNG/PDF

The preview server must be running at port 7720. Then run:

pip install playwright pillow && playwright install chromium

python assets/production/export-assets.py

Outputs all files to assets/production/exports/ — thumbnail PNG, HUD PNG, 3 Shorts card PNGs at 1080×1920, brand sheet PDF.

Go-Live Pipeline — Solo Cinematic

1

Read the Cinematic Brief First

ep01-cinematic-brief.html — all 9 skills output, hook selection, title selection, narrative arc, thumbnail concept. This is the strategy layer. Everything downstream follows it.

2

Film — Pure B-Roll, No On-Cam

Use ep01-shot-list.html. 34 shots across 5 acts. Priority 1: the door-open shot. VO recorded separately in a dead room. You operate camera. Script is locked before you pick up the camera.

3

Get Visual Direction Plan

Run /visual-director on the locked script. Edit the master cut to that plan.

4

Cut Shorts from Master

Follow ep01-shorts-plan.html. 7 clips. All timestamps and edit notes already written.

5

Export Brand PNGs

Run export-assets.py → thumbnail-ep01.png, hud-overlay.png, shorts-card-a/b/c.png. Already done for EP01.

6

Upload Episode

Use ep01-upload-package.html — everything pre-written. Paste title, description, chapters, tags. Upload thumbnail. Set Age-Restricted.

7

Update Website

Copy YouTube video ID → replace data-youtube-id in watch/index.html and watch/episode-01.html. Submit sitemap.xml.

8

Upload Shorts — Days 1–5

Follow the upload sequence in ep01-shorts-plan.html. Stagger over 5 days to extend discovery window.
